# Reflection Paper Lead Frontend
Selama pengerjaan proyek ini, saya menghadapi berbagai kendala teknis. Salah satunya adalah data dari backend yang tidak dapat ditampilkan pada frontend, serta masalah navigasi yang menyebabkan perpindahan antar halaman tidak berjalan sebagaimana mestinya. Selain itu, konfigurasi nginx.conf sempat mengalami error ketika dilakukan penambahan auth-service dan item-service, sehingga membutuhkan waktu yang cukup lama untuk proses troubleshooting dan perbaikannya.

Kendala lainnya adalah sering terjadinya conflict saat melakukan pull request antar anggota tim. Saya juga menghadapi permasalahan pada fitur dark mode, di mana perubahan tampilan hanya diterapkan pada bagian navbar, sedangkan komponen halaman lainnya tidak ikut menyesuaikan. Dari sisi infrastruktur, penggunaan Docker menjadi tantangan tersendiri karena merupakan pengalaman pertama saya menggunakannya. Proses instalasi Docker juga sempat terhambat akibat keterbatasan ruang penyimpanan pada perangkat yang digunakan.

Melalui berbagai kendala tersebut, saya memperoleh banyak pembelajaran, terutama mengenai pentingnya koordinasi dan komunikasi yang efektif dalam tim untuk mengurangi potensi conflict selama pengembangan. Selain itu, saya menyadari perlunya memperdalam pemahaman terkait teknologi seperti Docker dan arsitektur microservices agar proses implementasi dapat berjalan lebih lancar dan meminimalkan terjadinya kesalahan di kemudian hari.
